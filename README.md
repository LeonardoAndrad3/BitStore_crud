## Configuração de setup

- Clona o repositório no lugar que preferir;
- Instale as dependências:
```bash
npm install
```
- Inicie o servidor React:
```bash
npm start
```

- Não se preocupe com banco de dados ou as requisições com o mesmo, pois estão em nuvem.
